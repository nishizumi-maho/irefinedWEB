# Documentation

This directory keeps repository-side documentation that complements the GitHub wiki.

## Structure

- `changelogs/`: version-specific release notes and historical changelog files
- `research/`: larger analysis/reference documents that are useful to keep in-repo
- `wiki/`: source pages that are published to the GitHub wiki

## Primary Documentation Entry Points

- GitHub wiki: `https://github.com/nishizumi-maho/irefinedWEB/wiki`
- Root changelog: `../CHANGELOG.md`
- Releases: `https://github.com/nishizumi-maho/irefinedWEB/releases`

The repository README intentionally stays concise. Detailed feature, architecture, storage, privacy, troubleshooting, and build documentation lives in the wiki.
